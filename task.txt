Нужно написать консольную golang утилиту, не используя сторонние библиотеки и базы данных. 
На входе файл с поисковыми запросами, на выходе файл в формате tsv с уникальными запросами и их частотой. 
Например:

input.txt 

```
this
test 
asd
the
end
sad
this
is
asd
end
my
only
test
the 
```

output.tsv 
```
this 2 
test 2 
asd 2 
the 2 
end 2 
sad 1 
is 1 
my 1 
only 1 
```

Считаем, что одновременно в оперативную память влезает N уникальных серчей. N задаётся параметром. 
Для программы доступна безграничная файловая система.
